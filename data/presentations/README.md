# Presentations Directory

Place your presentation files here.

## Supported Formats

- **PDF** (`.pdf`) - PDF presentations
- **PowerPoint** (`.pptx`) - Microsoft PowerPoint files

## Example

```
presentations/
├── ml_intro_2024.pdf
├── data_science_basics.pptx
└── ai_overview_2024.pdf
```

## Notes

- Files will be automatically processed when you run the pipeline
- Large files may take longer to process
- Ensure files are readable and not corrupted
