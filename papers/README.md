## TReO Talks 2024

**Title**: Designing Real-Time Accessible Learning for Visually Impaired Students

**Conference**: TReO Talks 2024

**Paper**: [TReO_Talks_2024.pdf](TReO_Talks_2024.pdf)

**Abstract**: This paper presents a RAG-powered system for generating real-time accessible narratives from presentation documents, specifically designed to enhance learning experiences for visually impaired students. The system uses LangChain and FAISS to process 200+ presentations and generate contextual narratives with optimized performance for live educational deployments.
